F.A.Q.

"Will Calorie Counter affect individuals differently if they have genetic disorders?"

Yes, if the user has diabetes, Celiac disease, Crohn's disease, or other diseases that require irregular calorie intakes, then the user should contact their Physician before attempting to consume our recommended number of calories.

"How does Calorie Counter work exactly?"

Calorie Counter Calculates the users BMI (Body Mass Index), then based off how much their BMI deviates from the average BMI standard, Calorie Counter then predicts the users body type status (normal, overweight, underweight). Then from this information, calorie counter will recommend the number of calories based on their body Status. NormalCalorieIntake+-500 will determine weight loss/gain. For example, if the user is overweight, then we will recommend their NormalCalorieLimit -500. Therefore they will lose a pound a week. If their body status determines underweight, then recommendation is NormalCalorieLimit+500, so they will gain a pound a week. Lastly, normal body weight will result in NormalCalorieLimit, which will cause the user to remain the same weight.

"Is Calorie Counter gender specific?"

Yes, Calorie Counter is gender specific-hence why we ask for your gender input. BMI (Body Mass Index) has a different average for males and females.

"What's the point of the Calorie Counter Graph feature?"

The Calorie Counter graph feature is a user friendly feature that allows the user to track their progress of weight gain,weight loss, or weight consistency. It provides a basis that allows users to realize the impact that the calorieIntake has made on their weight (pounds) with respect to time (months).
